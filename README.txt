World Time
=============

INTRODUCTION
------------
This module is used to display the current time in a block based on the settings from a configuration form.

INSTALLATION
------------
Install the module using `drush en worldtime` command or from the Extend page in admin panel.

CONFIGURATION
-------------
If you go to `admin/config/regional/worldtime-settings`
you will see a simple form with three fields.

There are default values set in the form and you can update based on the preference.

Place the block `World Time Block` in any for the region from block layout page.

Clear the cache before reloading the pages.
